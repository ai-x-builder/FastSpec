# Loop Runner Artifact Templates

Use these templates when creating or updating Loop Runner implementation artifacts under `specs/<id>/`.

Read only the artifact template needed for the current action:

- `agent-assignments.md` for `AGENT_ASSIGNMENTS.json`
- `loop-state.md` for `LOOP_STATE.json`
- `trace-jsonl.md` for `TRACE.jsonl`
- `verify.md` for `VERIFY.md`
- `review.md` for `REVIEW.md`
- `report.md` for `REPORT.md`
